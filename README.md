# technobuddha.com

Technobuddha application

## Commands

| Command         | Description                                    |
| --------------- | ---------------------------------------------- |
| `yarn build`    | Build the application                          |
| `yarn dev`      | Build the application and watch for changes    |
| `yarn lint`     | Run ESLint to check the application for errors |
| `yarn test`     | Run unit tests                                 |
